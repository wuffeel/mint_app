# About the project

This is a package to use common features of Mint application 
across mobile and web versions.

# Features

- Clean Architecture
- Cross platform dependency injection, services, utilites, etc.
- Export modules for web and mobile

# Built With

- [Flutter](https://flutter.dev/) - Cross platform framework to build applications with single codebase
- [Firebase](https://firebase.google.com/) - Backend as a service
- [Flutter Bloc](https://bloclibrary.dev/) - State management library
- [GetIt](https://pub.dev/packages/get_it) & [Injectable](https://pub.dev/packages/injectable) - Dependency injection 
libraries

# Getting Started

#### Step 1:

Download or clone the repository by using link below:

```
https://github.com/wuffeel/mint_app
```

#### Step 2:

Resolve Flutter dependencies by running the following command:

```
flutter pub get
```
